Название прецедента: Create a database on the existing one.

Действующее лицо: Администратор.

Цель: создание базы данных, на основе открытой, с чистой таблицей "TimeTrack", 
  и таблицей "Person", в которой отсутствуют записи об уволенных работниках.

Предусловия: администратор вошёл в систему и открыл базу данных.

Главная последовательность:
  Администратор выбирает в главном меню пункт "Создать БД на основе существующей".
  Система выводит окно "Создание БД".
  Администратор выбирает место сохранения базы данных.
  Администратор указывает имя новой базы данных.
  Администратор нажимает кнопку создать.
  Система создаёт копию открытой базы данных, с новым именем. (закрывает старую БД)
  Система открывает новую базу данных.
  Система удаляет записи в таблице "TimeTrack".
  Система удаляет записи уволенных работников в таблице "Person".
  Система делает запись в журнале логирования.
  Система выводит окно сообщения о проделанной работе.
  Администратор нажимает кнопку "Да".
  Система возвращает фокус в главное окно программы. (Открыта новая БД)
  
Альтернативная последовательность: Администратор нажимает кнопку "Отмена" в окне "Создание БД".
    Система закрывает окно "Создание БД".
    Система возвращает фокус в главное окно приложения.
