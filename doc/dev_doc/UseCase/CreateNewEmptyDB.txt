Название прецедента: Create a new empty database.

Действующее лицо: Администратор.

Цель: создание новой незаполненной базы данных.

Предусловия: администратор вошёл в систему.

Главная последовательность:
  Администратор выбирает в главном меню пункт "Создать пустую базу данных".
  Система выводит на экран окно "Создание БД".
  Администратор выбирает место для сохранения базы данных. (по умолчанию рабочая папка программы).
  Администратор указывает имя файла базы данных. (по умолчанию TimeTrack.db)
  Администратор нажимает кнопку "Создать" в окне "Созддание БД".
  Система закрывает окно "Создание БД".
  Система создаёт новую базу данных с незаполненными таблицами.
  Система делает запись в журнал логирования.
  Система возвращает фокус в главное окно программы.
  
Альтернативная последовательность: Администратор нажимает кнопку "Отмена" в окне "Создание БД".
  Система закрывает окно файлового менеджера.
  Система возвращает фокус в главное окно программы.
