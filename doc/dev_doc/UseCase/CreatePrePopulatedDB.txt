Название прецедента: Create a pre-populated database.

Действующее лицо: Администратор.

Цель: создание базы данных с заполненными таблицами "Position" и "TypeWorkingTime".

Предусловия: администратор осуществил вход в систему.

Главная последовательность:
  Администратор выбирает в главном меню пункт "Создать предварительно заполненную БД".
  Система выводит на экран окно "Создание БД".
  Администратор выбирает место для сохранения базы данных. (по умолчанию рабочая папка программы)
  Администратор указывает имя файла базы данных. (по умолчанию TimeTrack.db)
  Администратор нажимает кнопку "Создать".
  Система закрывает окно "Создание БД".
  Система создаёт базу данных с заполненными таблицами "Position" и " TypeWorkingTime".
  Система создаёт запись в журнале логирования.
  Система возвращает фокус в главное окно программы.
  
Альтернативная последовательность:
  Администратор нажимает кнопку "Отмена".
  Система закрывает окно "Создание БД".
  Система возвращает фокус в главное окно программы.
