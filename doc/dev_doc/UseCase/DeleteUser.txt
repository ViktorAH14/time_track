Название прецедента: Delete user.

Действующее лицо: администратор.

Цель: удаление пользователя из системы.

Предусловия: администратор вошёл в систему.

Главная последовательность:
  Администратор выбирает в главном меню пункт "Удалить пользователя".
  Система выводит на экран окно "Удаление пользователя".
  Администратор выбирает в таблице запись для удаления.
  Администратор нажимает кнопку "Удалить".
  Система выводит предупреждающее сообщение.
  Администратор нажимает кнопку "Да" в окне предупреждающего сообщения.
  Система проверяет возможность удаления.
  Система удаляет пользователя.
  Система делает запись в журнал логирования.
  Система возвращает фокус в окно удаления пользователя.
  
Альтернативная последовательность: пользователь нажимает кнопку "Отменить" в окне предупреждения об удалении пользователя.
  Система возвращает фокус в окно удаления пользователя.
  
Альтернативная последовательность: администратор нажимает кнопку "Закрыть".
  Система закрывает окно удаления пользователя.
  Система возвращает фокус в главное окно приложения.
    
Альтернативная последовательность: удаляемый пользователь, является последним администратором в системе.
  Система показывает окно с сообщением о невозможности удаления данного пользователя.
  Администратор нажимает кнопку "Да" в сообщении.
  Система возвращает фокус в окно удаления пользователя.
