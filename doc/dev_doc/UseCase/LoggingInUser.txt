Название прецедента: Logging in (Вход в систему).

Действующее лицо: User.

Цель: войти в систему.

Предусловия: на экран выведено окно входа в систему, пользователь зарегистрирован в системе администратором.

Главная последовательность:
  Пользователь вводит в поле "Имя" свой Login.
  Пользователь вводит в поле "Пароль" свой пароль.
  Пользователь нажимает кнопку "Войти".
  Система проверяет правильность введённых данных.
  Система делает запись в журнале логирования.
  Система открывает главное окно программы.

Альтернативная последовательность: имя или пароль пользователя введены не правильно
  Система делет запись в журнале логирования.
  Система выводит окно с сообщением о том, что данные введены не правильно и сделана соответствующая запись в журнал логирования.
  Пользователь нажимает кнопку "Да"
  Система снова показывает окно с полями для ввода имени и пароля.
  
Альтернативная последовательность: администратор нажимает кнопку "Отмена"
  Выход из программы.
